Story Flow: Multi-Intent IVA Resolution Journey

Target + Genesys Cloud AI Demo
Multi-Intent Resolution Flow

Scenario

A Target customer calls about their Target Circle Card. They first request a balance inquiry, then report a fraudulent charge. The Genesys Cloud Virtual Agent expresses empathy, walks through verification steps, and offers to cancel the card and send a replacement. When the customer requests expedited shipping due to upcoming Black Friday, the Virtual Agent escalates to a live Target agent. Genesys Agent Copilot summarizes the interaction and assists the agent with Knowledge Articles, Next Best Action recommendations, and Disposition suggestions.

Flow 1: Virtual Agent Interaction

Customer Call

Customer calls Target support line via voice channel.

Virtual Agent Greeting

Genesys Cloud Virtual Agent greets and identifies the customer.

Balance Request

Customer asks for Target Circle Card balance; VA retrieves and reads back balance.

Fraud Concern

Customer reports a fraudulent charge; VA expresses empathy and explains verification steps.

Escalation Trigger

VA escalates to live Target agent for further handling.

Expedite Request

Customer requests expedited delivery before Black Friday.

Cancellation Offer

VA offers to cancel card and send replacement; customer accepts.

Flow 2: Live Agent & Copilot Experience

Agent Handoff

Genesys Agent Copilot summarizes the conversation for the Target agent.

Contextual Guidance

Copilot surfaces Knowledge Articles, Next Best Actions, and Agent Checklists in real time.

Disposition Suggestion

Post-call, Copilot recommends the best Disposition to classify the interaction.
