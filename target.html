Story Flow: Multi-Intent IVA Resolution Journey

Target + Genesys Cloud AI Demo
Multi-Intent Resolution Flow

Scenario

A Target customer, Paul, calls to report a lost Target Circle Card. He requests a balance inquiry and replacement card delivery. The Genesys Cloud Virtual Agent expresses empathy and begins to processes both of Paul's intents. When the customer requests expedited shipping due to upcoming Black Friday, the Virtual Agent provides a Knowledge response, then escalates to a live Target agent. Genesys Agent Copilot summarizes the interaction and assists the agent with real-time Knowledge Articles recommendations, and Disposition suggestions.

Flow 1: Virtual Agent Interaction

Customer Call

Paul calls Target support line via voice channel.

Virtual Agent Greeting

Genesys Cloud Virtual Agent greets and identifies the Paul

Multi-Intent Request

Paul requests his Circle Card balance and replacement card delivery

Balance Inquiry Handling

Virtual Agent provides the balance and verifies no fraudulent charges.

Escalation Trigger

Virtual Agent escalates to a live Target agent for further handling.

Expedite Request

Paul requests expedited delivery to arrive before Black Friday.

Replacement Card Delivery Handling

Virtual Agent offers to cancel Circle Card and send replacement; Paul accepts.

Flow 2: Live Agent & Copilot Experience

Agent Handoff

Genesys Agent Copilot summarizes the conversation for the Target agent.

Contextual Guidance

Copilot surfaces Knowledge Articles in real time.

Predictive Wrap-Up

Post-call, Copilot recommends the best Disposition to classify the interaction.
