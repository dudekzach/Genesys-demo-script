Story Flow: Multi-Intent IVA Resolution Journey

Genesys turns every Target interaction into loyalty, efficiency and revenue through real-time orchestration of people, data and AI.

Scenario

A Target customer, Paul Dram, calls to report a lost Target Circle Card. He requests a balance inquiry and replacement card delivery. The Genesys Cloud Virtual Agent, named Christina, expresses empathy and begins processing both of Paul's intents. When Paul requests expedited shipping due to upcoming Black Friday, the Virtual Agent provides a Knowledge response, then escalates to a live Target agent. Genesys Agent Copilot summarizes the interaction and assists the agent with Knowledge Articles and Disposition recommendations in real time.

Flow 1: Virtual Agent Interaction

Customer Call

Paul calls Target’s customer service line via voice channel.

Virtual Agent Greeting

Christina, the Genesys Cloud Virtual Agent, greets and asks how she can help today.

Balance Inquiry

Paul asks to check his balance to verify no fraudulent charges.

Fraud Review

Virtual Agent confirms recent transactions and validates the last charge.

Replacement Card Request

Christina offers to issue a replacement card for Paul’s lost Circle Card.

Address Confirmation

Virtual Agent verifies Paul’s mailing address and processes the replacement order.

Expedited Delivery Request

Paul requests expedited shipping because Black Friday is approaching.

Escalation Trigger

Christina offers to transfer Paul to a live agent to review expedited shipping options.

Flow 2: Live Agent & Copilot Experience

Agent Handoff

Genesys Agent Copilot summarizes Paul’s virtual agent interaction for the live Target agent.

Contextual Guidance

Copilot provides real-time guidance and Knowledge Articles related to expedited card replacement.

Disposition Suggestion

After resolution, Copilot recommends the appropriate disposition code for the interaction.
